## Prompt

Build a modern, premium, minimal student productivity dashboard web app called "FocusFlow".

Tech stack:
- React
- Tailwind CSS
- Dark theme only
- Clean SaaS style UI

Design requirements:
- Very minimal
- Professional
- Lots of whitespace
- Soft shadows
- Rounded 2xl cards
- Modern typography (Inter font)
- Sidebar layout like Notion or Linear
- No bright colors
- Use deep dark background

Color palette:
- Background: #0B0F14
- Cards: #111827
- Borders: #1F2937
- Text primary: #F9FAFB
- Text secondary: #9CA3AF
- Accent: #6366F1

Features:
- Sidebar navigation
- Dashboard page
- Add daily tasks (todo list with checkbox)
- Pomodoro timer (25/5 minutes)
- Study hours tracker
- Coding hours tracker
- Workout log
- Daily streak counter
- Weekly progress charts
- Local storage for saving data
- Fully responsive

Components:
- Stats cards
- Timer card
- Chart section
- Task manager
- Clean buttons
- Icons only where needed

Make it look like a professional SaaS product, not a student project.

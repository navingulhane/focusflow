# FocusFlow

FocusFlow is a modern productivity dashboard built for students who want to manage their time, stay disciplined, and track daily progress in one clean interface.

It combines task management, focus sessions, study tracking, workout logs, and analytics into a minimal SaaS-style dashboard designed to improve consistency and performance.

## Features

- Daily task manager (add, complete, delete tasks)
- Pomodoro focus timer (25/5 sessions)
- Study hours tracker
- Coding time tracker
- Workout log
- Daily streak counter
- Weekly progress analytics
- Local storage persistence
- Responsive design
- Minimal dark theme interface

---
## Tech Stack

- React
- Tailwind CSS
- JavaScript
- Local Storage
- Vercel (Deployment)

---

## Design Philosophy

FocusFlow follows a clean and distraction-free design system inspired by modern SaaS products.

- Dark theme for reduced eye strain
- Minimal layout with generous spacing
- Card-based dashboard
- Soft shadows and rounded corners
- Professional typography
- No visual clutter

The goal is to help students focus on execution rather than complexity.
